Pilas y colas ¿Qué son?

Las pilas y colas son estructuras de datos que se utilizan generalmente para simplificar ciertas operaciones de programación. Estas estructuras pueden implementarse mediante arrays o mediante listas enlazadas. Estas estructuras de datos nos permiten tener una mejor gestión de información, mediante estas podremos insertar elementos o eliminarlos sin necesidad de hacer modificaciones en el código durante la ejecución del programa. Esto nos dará un rendimiento más optimo y una mejor manipulación de datos.
